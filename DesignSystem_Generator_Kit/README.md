# Design System Generator Kit - 使用說明書

這是一個獨立的設計系統自動化工具包，旨在幫助 UI/UX 設計師快速產出符合開發規範的 Tokens 與 Figma AI 繪圖指令。

## 📁 資料夾結構
- `GEMINI.md`: 核心行為準則，確保 AI 使用台灣繁體中文與專業用語。
- `skills/design-system-gen.md`: Skill 的邏輯定義檔（AI 的大腦）。
- `templates/DESIGN_SYSTEM_BRIEF.md`: 設計需求填寫表（你的輸入）。

## 🚀 安裝與啟動步驟

### 第一步：安裝到新專案
1. 將 `GEMINI.md` 複製到新專案的 `.gemini/` 資料夾下。
2. 將 `skills/design-system-gen.md` 複製到新專案的 `.gemini/skills/` 資料夾下。
3. 將 `templates/DESIGN_SYSTEM_BRIEF.md` 複製到你的專案目錄中。

### 第二步：設定需求
1. 開啟 `DESIGN_SYSTEM_BRIEF.md`。
2. 填入你的品牌名稱、主色 Hex Code、字體偏好與風格描述。

### 第三步：執行指令
在 Gemini CLI 中輸入以下任意指令：
- 「我填好設計範本了，請幫我產出設計規範。」
- 「根據目前的設計需求，製作一套新的設計系統。」

## 🎁 產出成果
執行後，專案根目錄會出現以下檔案：
1. `[品牌]_tokens.json`: 可匯入 Token Studio 的資料檔。
2. `[品牌]_GUIDELINE.md`: 給開發團隊看的中文設計指南。
3. `[品牌]_FIGMA_PROMPT.txt`: 給 Figma AI (Make Design) 的繪圖指令。

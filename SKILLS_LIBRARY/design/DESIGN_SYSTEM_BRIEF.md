# Design System Brief (需求範本)

## 1. 專案基本資訊
- **Project Name**: Klook_Style_System
- **Description**: 一個充滿活力、以旅遊探索為核心的設計系統，模仿 Klook 的品牌視覺風格。強調高飽和度的主色、圓潤親切的介面元素，以及清晰的資訊層級，專為行動端優先的旅遊體驗打造。

## 2. 品牌色彩 (Brand Colors)
- **Primary Color (Hex)**: #FF5B00
  - *說明*: Klook 經典活力橘，用於主要按鈕、圖示亮點與品牌識別。
- **Secondary Color (Hex)**: #1C1C1C
  - *說明*: 深黑色，用於主要標題與強調文字，提供高對比度。
- **Background Color (Hex)**: #F5F5F5
  - *說明*: 淺灰色背景，讓白色卡片能浮現出來。

## 3. 字體與排版 (Typography)
- **Font Family (中/英)**: PingFang TC / Roboto
- **Style Keywords**: Modern (現代), Energetic (活力), Legible (易讀), Friendly (親切)
- **Base Size**: 16px

## 4. 介面物理特性 (UI Properties)
- **Border Radius**: 12px
  - *說明*: 用於卡片與彈窗，創造親切感。按鈕可採用 8px 或全圓角 (Pill)。
- **Spacing Scale**: 8px (Base unit)
- **Shadow Style**: Soft, Diffused (柔和擴散)
  - *說明*: 用於提升卡片的層次感，使其在淺灰背景上更立體。

## 5. 特殊需求 (Specific Requirements)
- 按鈕必須高對比且顯眼 (High contrast CTA)。
- 圖示風格應為圓潤的線條 (Rounded Line Icons)。
- 強調圖片展示區域 (Image-centric layout)。
- 必須包含行動端底部導航列 (Bottom Tab Bar)。
